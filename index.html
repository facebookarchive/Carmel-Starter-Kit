Carmel Starter Kit Samples

Welcome to the Carmel WebVR Starter Kit. Here you will find a set of samples which helps introduce you to the Web VR 1.1 API and how to use it to create experiences for the Carmel Developer Preview.
This page is best launched from your Android browser of choice. Next follow each link to either view your content in the Carmel Developer Preview using the ovrweb protocol on Android or launch it in your existing 2D browser to see how it will look outside of VR.
For more information on each sample you can review our full documentation online at the Oculus Developer Center.
As we make changes you can download the latest version of our samples from the Oculus Developer Site. We will also keep our live samples updated with any changes.
Please provide any feedback on the samples, documentation or Carmel.
